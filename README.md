# Data Structures and Algorithms 2  
### Programming Tasks (1–16)

This repository contains my solutions of algorithmic programming tasks completed as part of the **[Data Structures and Algorithms 2](https://opiskelijanopas.tuni.fi/en/tampere-university/curriculum/course-units/uta-ykoodi-25936?year=2025)** course at Tampere University
The course focuses on implementing intermediate‑level algorithms efficiently in **C++**, **Java**, or **Python**.  
All 16 programming tasks are included in this repository.

---

## Course Overview

The course covers key algorithmic techniques and data structures, with emphasis on practical implementation.  
Topics include:

- Minimum spanning trees  
- Shortest paths with and without negative weights  
- Cycle detection and topological sorting  
- Maximum flow and bipartite matching  
- Segment trees and interval queries  
- Efficient string sorting  
- String matching (MP, BM, Aho–Corasick)  
- Tries and suffix structures  
- Regular expression matching with automata  
- Approximate string comparison (LCS, edit distance)

## Automated Grading (WETO)

The course uses an automated grading system (WETO).  
Key points to understand when working with the tasks:

- Code must compile (for C++/Java) before tests run.  
- Each task is evaluated using multiple test cases.  
- Public tests show diff‑style feedback for incorrect output.  
- Private tests are used during final grading.  
- Full points typically require passing **all** test cases.  
- Feedback may include:
  - Incorrect output lines  
  - Missing newline characters  
  - Crashes or time/memory limit issues  

Example diff feedback:

    -Algorithm 
    +Algorithm
     course
    -5 ECTS
    \ No newline at end of file
    +5 ECTS
